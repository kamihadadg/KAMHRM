# KAMHRM - سامانه مدیریت منابع انسانی

یک پلتفرم جامع برای مدیریت منابع انسانی، پرسشنامه‌ها و سازمان‌بندی تیم‌ها

## نمای کلی پروژه

KAMHRM یک سیستم مدیریت منابع انسانی مدرن و کامل است که برای سازمان‌ها و شرکت‌ها طراحی شده‌است. این سیستم امکانات جامعی برای مدیریت کارمندان، قراردادها، انتصاب‌ها، پرسشنامه‌ها و فیدبک‌های کارمندان را فراهم می‌کند.

## ویژگی‌های اصلی

### مدیریت کاربران و احراز هویت
- سیستم ثبت‌نام و ورود امن
- مدیریت نقش‌ها و دسترسی‌ها
- احراز هویت مبتنی بر JWT
- تغییر رمز عبور محفوظ

### مدیریت منابع انسانی
- مدیریت معلومات کارمندان
- ثبت و دنبال‌کاری قراردادهای کار
- مدیریت انتصاب‌ها و جابجایی‌های کارمندان
- تعریف و سازماندهی پست‌ها و بخش‌ها

### پرسشنامه‌ها و فیدبک
- ایجاد و انتشار پرسشنامه‌های جامع
- تحلیل نتایج و پاسخ‌های کارمندان
- سیستم نظرات و فیدبک‌های سازنده
- گزارش‌گیری دقیق از نتایج

### نمایش سازمانی
- نمایش تعاملی نمودار سازمان‌شناسی
- مشاهده ساختار سلسله‌مراتبی سازمان
- جستجو و فیلتر در ساختار سازمان

### مدیریت تنظیمات
- تنظیمات حساب کاربری
- تصویر پروفایل و اطلاعات شخصی
- مدیریت تنظیمات سیستم

## پشته فناوری

### Backend
- **Framework**: NestJS (Node.js)
- **زبان**: TypeScript
- **احراز هویت**: JWT (JSON Web Tokens)
- **Validation**: Class Validator
- **Database**: رابطه‌ای (پیکربندی شده)

### Frontend
- **Framework**: Next.js 15+
- **زبان**: TypeScript
- **استایل**: CSS/Tailwind
- **Client-Side**: React 19+
- **Communication**: API REST

### ابزارهای توسعه
- ESLint برای کیفیت کد
- Jest برای تست‌های واحد
- TSConfig برای تنظیمات TypeScript

## ساختار پروژه

### Backend (`/backend`)
سرور NestJS که مسئول پردازش تمام منطق کسب‌وکار و مدیریت داده‌ها است
- ماژول احراز هویت: مدیریت ورود و کنترل دسترسی
- ماژول منابع انسانی: قراردادها و انتصاب‌ها
- ماژول پرسشنامه: ایجاد و تحلیل پرسشنامه‌ها
- فایل‌های اپلود: ذخیره تصاویر پروفایل

### Frontend (`/frontend`)
اپلیکیشن Next.js که رابط کاربری تعاملی را فراهم می‌کند
- صفحه پیش‌فرض: نقطه ورودی اپلیکیشن
- پنل مدیریتی: مدیریت کاربران و تنظیمات
- داشبورد: خلاصه‌ای از اطلاعات مهم
- بخش پرسشنامه‌ها: مدیریت و تجزیه‌تحلیل پرسشنامه‌ها
- بخش سازمان: نمایش ساختار سازمانی
- بخش فیدبک: دریافت و مدیریت فیدبک‌ها

## شروع سریع

### پیش‌نیازها
- Node.js نسخه 18 یا بالاتر
- npm یا yarn برای مدیریت بسته‌ها
- دسترسی به پایگاه‌داده (PostgreSQL یا دیگر دستگاه‌های پشتیبانی‌شده)

### نصب
1. مخزن را کلون کنید
2. برای backend و frontend به ترتیب وارد هر دایرکتوری شوید
3. وابستگی‌های پروژه را نصب کنید
4. متغیرهای محیطی را تنظیم کنید
5. سرویس backend و frontend را اجرا کنید

### اجرای سرویس‌ها
- **Backend**: روی درگاه 3000 (یا درگاه پیکربندی‌شده) اجرا می‌شود
- **Frontend**: روی درگاه 3001 یا 3002 اجرا می‌شود

## مدول‌های اصلی

### Authentication Module
مسئول احراز هویت کاربران و کنترل دسترسی سطح سیستم

### HR Module
مدیریت کل جنبه‌های منابع انسانی از قراردادها گرفته تا انتصاب‌ها

### Survey Module
سیستم جامع پرسشنامه‌سازی با قابلیت‌های تجزیه‌تحلیل

### User Management
مدیریت اطلاعات کاربران، نقش‌ها و دسترسی‌ها

## نقش‌های کاربری

سیستم از چندین سطح دسترسی پشتیبانی می‌کند:
- **مدیر سیستم**: دسترسی کامل به تمام بخش‌ها
- **مدیر منابع انسانی**: مدیریت کارمندان و موارد متعلقه
- **کارمند**: دسترسی به اطلاعات شخصی و شرکت در پرسشنامه‌ها

## ویژگی‌های امنیتی

- احراز هویت مبتنی بر JWT
- رمزگذاری امن رمز عبور
- کنترل دسترسی مبتنی بر نقش
- اعتبارسنجی ورودی
- محافظت از CORS

برای توسعه ویژگی‌های جدید:
